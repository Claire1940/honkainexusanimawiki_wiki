<Section icon={Shield} title="Plataformas compatibles y requisitos del sistema">

En su lanzamiento, Honkai: Nexus Anima se centrará en gran medida en la accesibilidad multiplataforma. El juego está optimizado para PC, iOS y dispositivos Android, lo que permite a los jugadores alternar sin problemas entre el juego en PC y en dispositivos móviles. Aunque no se ha confirmado un lanzamiento en consolas (como PlayStation 5) para el primer día, los títulos anteriores sugieren que una versión para consolas podría llegar aproximadamente seis meses después del lanzamiento.

Para ejecutar la Evolution Test actual sin problemas, tu sistema debe cumplir con los siguientes requisitos de hardware:

| Componente de hardware | Especificaciones mínimas | Especificaciones recomendadas |
| :--- | :--- | :--- |
| **Sistema operativo** | Windows 10 (64 bits) | Windows 10 / 11 (64 bits) |
| **Procesador (CPU)** | Intel Core i5-10400 / AMD Ryzen 5 3600 | Intel Core i5-11400F / AMD Ryzen 5 3600X |
| **Tarjeta gráfica (GPU)** | GTX 1060 / AMD Radeon RX 580 | RTX 2060 / AMD Radeon RX 5600 XT |
| **Memoria del sistema (RAM)** | 8 GB de RAM | 12 GB de RAM |
| **Espacio de almacenamiento** | 30 GB de espacio disponible | Se prefiere SSD de 30 GB |

<Callout type="info" title="Requisitos para móviles">
Para iOS, se recomienda un chip Apple A14 Bionic (iPhone 12) o superior. Para Android, un procesador Snapdragon 888 o equivalente es ideal para mantener 60 FPS durante escenas de combate intensas.
</Callout>

</Section>
