<Section icon={Target} title="PC Launcher Installation and Troubleshooting Guide">

Setting up the launcher for the Evolution Test can occasionally present technical hurdles due to system permissions or scaling conflicts. Follow these structured steps to install the game client and resolve common launcher bugs.

<Steps>
  <Step num="1" title="Download the Official Client">
    Log into your authorized HoYoverse account on the official website and download the game installer. Ensure you have at least 35GB of free space on your targeted drive.
  </Step>
  <Step num="2" title="Configure High DPI Scaling">
    If the launcher interface scales abnormally beyond your monitor bounds, right-click the desktop icon, go to **Properties** > **Compatibility** > **Change high DPI settings**, check **Override high DPI scaling behavior**, and click **Apply**.
  </Step>
  <Step num="3" title="Enable Windows Update Services">
    If you receive error code 1058 regarding missing plugins, open the Windows Run window (Win + R), type `services.msc`, locate **Windows Update**, set the startup type to **Manual**, apply changes, and restart the launcher.
  </Step>
  <Step num="4" title="Verify Network and Permissions">
    For update failure loops, disable active proxies, check your hosts file to ensure connections are not blocked, and run the launcher executable as an Administrator.
  </Step>
</Steps>

<CardGrid cols={3}>
  <Card title="Error: Interface Scaling">
    - **Symptom**: Launcher cut off or oversized.

  </Card>
  <Card title="Error: Plugin Failure (1058)">

  </Card>
</CardGrid>
