<Section icon={Shield} title="対応プラットフォーム＆動作環境">

リリース時、『崩壊：ネクサスアニマ』はクロスプラットフォームでのプレイアビリティに重点を置きます。ゲームはPC、iOS、Androidデバイス向けに最適化されており、プレイヤーはデスクトップとモバイルの間でシームレスにプレイを切り替えることができます。PlayStation 5などのコンソール版のリリースは初日には確認されていませんが、過去のタイトルから判断すると、リリースから約半年後にコンソールへの移植が行われる可能性があります。

現在のエボリューションテストを快適に動作させるには、システムが以下の動作環境を満たしている必要があります：

| ハードウェア構成 | 必要動作環境 | 推奨動作環境 |
| :--- | :--- | :--- |
| **OS** | Windows 10 (64ビット) | Windows 10 / 11 (64ビット) |
| **プロセッサー (CPU)** | Intel Core i5-10400 / AMD Ryzen 5 3600 | Intel Core i5-11400F / AMD Ryzen 5 3600X |
| **グラフィックボード (GPU)** | GTX 1060 / AMD Radeon RX 580 | RTX 2060 / AMD Radeon RX 5600 XT |
| **メモリ (RAM)** | 8 GB RAM | 12 GB RAM |
| **ストレージ容量** | 30 GBの空き容量 | 30 GBのSSD推奨 |

<Callout type="info" title="モバイル端末の動作環境">
iOSの場合、Apple A14 Bionicチップ（iPhone 12）以上が推奨されます。Androidの場合、負荷の高い戦闘シーンで60 FPSを維持するには、Snapdragon 888または同等以上のプロセッサーが理想的です。
</Callout>
